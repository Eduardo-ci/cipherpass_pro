    Aplicación Principal (Core)

    Esta sección detalla las clases y métodos principales utilizados en main.py para la criptografía, el análisis de seguridad y la interfaz gráfica.

    CryptoManager

    main.CryptoManager

    Gestiona la clave de cifrado maestra local para los diccionarios Diceware.

    Esta clase se encarga de crear o recuperar una clave simétrica de Fernet almacenada en el directorio de configuración del usuario.

    get_cipher_suite() staticmethod

    Obtiene la suite criptográfica para cifrado y descifrado local.

    Returns:

    Name Type Description
    Fernet Fernet

    Instancia de Fernet inicializada con la clave local.

    generate_password(length, min_nums, min_specs, use_upper, use_lower, use_nums, use_syms, avoid_amb)

    Genera una contraseña criptográficamente segura basada en restricciones.

    Parameters:

    Name Type Description Default
    length int

    Longitud total de la contraseña.

    		 required
    min_nums int

    Cantidad mínima garantizada de números.

    		 required
    min_specs int

    Cantidad mínima garantizada de caracteres especiales.

    		 required
    use_upper bool

    Si es True, incluye mayúsculas (A-Z).

    		 required
    use_lower bool

    Si es True, incluye minúsculas (a-z).

    		 required
    use_nums bool

    Si es True, incluye números (0-9).

    		 required
    use_syms bool

    Si es True, incluye símbolos definidos en DEFAULT_SYMBOLS.

    		 required
    avoid_amb bool

    Si es True, excluye caracteres ambiguos (l, 1, I, 0, O).

    		 required

    Returns:

    Name Type Description
    str str

    La contraseña generada, o una cadena vacía si no hay pool de caracteres.
