# Bienvenido a la documentación de CipherPass PRO 🔐

CipherPass es una potente y segura aplicación de escritorio diseñada para generar, validar y proteger credenciales criptográficas. Construida con **Python** y **Qt (PySide6)**, se enfoca en la privacidad *offline-first*.

## Estructura de la Documentación

En este sitio encontrarás toda la información técnica necesaria para entender, mantener y escalar el proyecto:

* **Inicio:** Esta página de bienvenida.
* **Arquitectura de Seguridad:** Detalles sobre la criptografía y seguridad offline-first de CipherPass.
